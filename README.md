# iesturnotarde

¿Alguna vez entraste a una habitación y olvidaste por completo qué ibas a buscar? No es un problema de memoria, sino un fenómeno psicológico real llamado "Efecto de Umbral".

Nuestro cerebro segmenta la realidad por contextos. Cuando cruzamos una puerta, el cerebro marca un "final de escena" y archiva la información de la habitación anterior para liberar espacio y procesar el nuevo entorno. Al cruzar el marco de la puerta, los pensamientos que tenías hace un segundo se guardan en una carpeta mental distinta, lo que genera ese vacío repentino de: "¿A qué venía yo?".
